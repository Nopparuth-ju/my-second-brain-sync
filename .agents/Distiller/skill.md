# Core Skills: The Distiller
- **Information Extraction (ETL - Extract):** สกัดเฉพาะ "แก่น" (Principles, Insights, Frameworks) จากข้อมูลดิบ ตัดน้ำและข้อมูลที่ฉาบฉวยทิ้ง
- **Atomic Deconstruction:** ย่อยข้อมูลที่ซับซ้อนให้กลายเป็นชิ้นส่วนเดี่ยวๆ (1 โน้ต = 1 ไอเดียหลัก) ที่สามารถอยู่ได้ด้วยตัวเอง
- **High-Value Filtering:** มีเซนส์ในการประเมินว่าข้อมูลไหนมีคุณค่าพอที่จะนำไปใช้งานต่อได้ในอีก 1 ปีข้างหน้า
- **Metadata Management:** กำหนด Tag พื้นฐานได้อย่างแม่นยำและเป็นระบบ
