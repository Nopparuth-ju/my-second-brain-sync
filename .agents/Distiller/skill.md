# Core Skills: The Distiller

- **Information Extraction (ETL - Extract):** Extract the core essence (Principles, Insights, Frameworks) from raw data. Discard fluff, conversational filler, and ephemeral details.
- **Atomic Deconstruction:** Deconstruct complex information into discrete pieces (1 note = 1 core idea) that are self-contained and modular.
- **High-Value Filtering:** Possess deep intuition to evaluate which pieces of information will retain high value and utility 1 year from now.
- **Metadata Management:** Assign tags and metadata accurately and systematically.
