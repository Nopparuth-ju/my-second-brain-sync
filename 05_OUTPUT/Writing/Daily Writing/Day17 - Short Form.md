
เวลาที่เราพูดถึงการสร้าง Data Pipeline อยากให้ลองนึกภาพถึง การ 'เตรียมวัตถุดิบในครัว' นะครับ
ก่อนที่เราจะได้อาหารจานที่อร่อย เราต้องไปหาซื้อวัตถุดิบจากแหล่งต่างๆ นำมาล้างเศษดิน ปอกเปลือก และหั่นให้พร้อมไปปรุงต่อ ซึ่งขั้นตอนนี้เป็นขั้นตอนที่กินพลังงานมากที่สุด และเปรียบเสมือนการทำ Data Pipeline 
ส่วนการนำวัตถุดิบเหล่านั้นไปลงกระทะ ปรุงรส และตกแต่งจานให้สวยงาม เปรียบเสมือนการสร้าง Dashboard หรือการสร้าง AI Model ครับ
ถ้าวัตถุดิบตั้งต้นเราคุณภาพต่ำ หรือทำความสะอาดได้ไม่ดีพอ ต่อให้เราใช้กระทะราคาแพง หรือตกแต่งจานสวยหรูแค่ไหน อาหารจานนั้นก็กินไม่ได้อยู่ดี (เหมือนข้อมูลที่ผิดเพี้ยน ทำให้ตัดสินใจผิดพลาด)

นี่จึงเป็นเหตุผลสำคัญที่เราต้องขออนุมัติเวลาและงบประมาณ เพื่อวางรากฐาน Data Pipeline ให้แข็งแรงมั่นคง ก่อนที่เราจะรีบกระโดดไปโฟกัสเรื่องการทำ Dashboard หรือ AI Model ครับ
